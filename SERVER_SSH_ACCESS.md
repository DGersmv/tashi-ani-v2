# Подключение к серверу 89.111.154.8 по SSH

Вход только по ключу, пароль отключён.

---

## С вашего основного компьютера (где создан ключ)

**Команда для входа:**

```cmd
ssh -i C:\Users\DGer\.ssh\tashi_ani root@89.111.154.8
```

В CMD или PowerShell. Пароль не запрашивается — вход по ключу `tashi_ani`.

---

## С другого компьютера

Есть два варианта.

### Вариант А: Использовать тот же ключ

1. Скопируйте с основного ПК на другой компьютер **два файла** (по USB, облаку и т.п.):
   - `C:\Users\DGer\.ssh\tashi_ani` (приватный ключ, без расширения)
   - `C:\Users\DGer\.ssh\tashi_ani.pub` (можно не копировать, он нужен только для добавления на сервер)

2. На другом компьютере положите ключ, например, в:
   - Windows: `C:\Users\<ИмяПользователя>\.ssh\tashi_ani`
   - Создайте папку `.ssh`, если её нет.

3. Подключитесь (подставьте **свой** путь к ключу и имя пользователя Windows):

   **Windows (CMD/PowerShell):**
   ```cmd
   ssh -i C:\Users\ИМЯ_ПОЛЬЗОВАТЕЛЯ\.ssh\tashi_ani root@89.111.154.8
   ```

   **macOS / Linux:**
   ```bash
   ssh -i ~/.ssh/tashi_ani root@89.111.154.8
   ```

Важно: храните `tashi_ani` в надёжном месте и никому не отдавайте — это полный доступ к серверу.

---

### Вариант Б: Создать новый ключ на другом компьютере

1. **На другом компьютере** создайте ключ:

   **Windows (CMD):**
   ```cmd
   mkdir %USERPROFILE%\.ssh 2>nul
   ssh-keygen -t ed25519 -f %USERPROFILE%\.ssh\tashi_ani -N ""
   type %USERPROFILE%\.ssh\tashi_ani.pub
   ```
   Скопируйте выведенную строку (начинается с `ssh-ed25519 ...`).

   **macOS / Linux:**
   ```bash
   ssh-keygen -t ed25519 -f ~/.ssh/tashi_ani -N ""
   cat ~/.ssh/tashi_ani.pub
   ```
   Скопируйте выведенную строку.

2. **Добавьте этот ключ на сервер.**  
   Зайдите на сервер с основного ПК (или с того, где уже есть доступ):

   ```cmd
   ssh -i C:\Users\DGer\.ssh\tashi_ani root@89.111.154.8
   ```

   На сервере выполните:

   ```bash
   nano ~/.ssh/authorized_keys
   ```
   Вставьте **новую** строку (из другого компьютера) в конец файла. Сохраните: Ctrl+O, Enter, Ctrl+X.

3. **С другого компьютера** подключайтесь своим ключом:

   ```cmd
   ssh -i путь\к\tashi_ani root@89.111.154.8
   ```
   или
   ```bash
   ssh -i ~/.ssh/tashi_ani root@89.111.154.8
   ```

## Краткая шпаргалка

| Где подключаетесь | Команда |
|-------------------|--------|
| Основной ПК (DGer) | `ssh -i C:\Users\DGer\.ssh\tashi_ani root@89.111.154.8` |
| Другой Windows | `ssh -i C:\Users\ИМЯ\.ssh\tashi_ani root@89.111.154.8` |
| macOS / Linux | `ssh -i ~/.ssh/tashi_ani root@89.111.154.8` |

Пароль при входе не используется. Если ключ потерян — войти будет нельзя; доступ тогда только через консоль хостера (VNC / rescue).
